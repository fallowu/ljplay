# ljplay

a website that develop by [vue](https://github.com/vuejs/vue) and [nuxt](https://github.com/nuxt/nuxt.js), and 
deploy to git pages 

## init
```bash
git clone ...
yarn install
or
npm install 
```

## dev
```bash
yarn dev
```

## ssr 
```bash
yarn build && yarn start
```

## static website deploy to git pages
```bash
yarn deploy
```
